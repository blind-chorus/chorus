# Figma plugin

> Planned. Not yet implemented.

A Figma plugin that imports the Chorus design system into Figma — tokens as variables, and (later) components and icons as library assets. Consumes `@blind-chorus/tokens`, `@blind-chorus/components`, and `@blind-chorus/icons` via npm workspaces, the same way `apps/docs` does.

Until the first version lands, this folder is intentionally empty.
